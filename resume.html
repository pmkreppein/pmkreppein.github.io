I am currently taking courses on a website called Treehouse. It's a great website for learning how to code, and is responsible for teaching most of what I know about coding. Treehouse allows students to earn points based on how many courses you take and pass. The more points you have in a topic, the more courses you've successfully completed.

Below is a dynamicaly-updating pie chart of my progress. It is built using the super-awesome JavaScript charting library called ChartJS. The data backing the chart is JSON data fetched from Treehouse using a jQuery $.ajax() call.

A note about the colors for the pie chart: they are automagically generated using a script I wrote, so if some of them clash (or sections of the pie use the same color) don't panic. Just refresh the page, and the pie chart will be in a whole different set of colors. Why not give it a whirl now? You can also hover over the sections of the pie chart for some handy tooltips.
